# 447-Project
Williams Industry Project

How to set up project:

Download Expo Go on iPhone

Clone repository into VSCode

npm install

npx expo start - to run the application

For running on expo go - scan the qr code generated in the terminal to open it on the app
For running on browser - click w while in the terminal

# Tech Stack
- Tailwind CSS with NativeWind to be compatible with React Native
- React Native for developing the application
- Expo for running it on a mobile application for testing purposes
